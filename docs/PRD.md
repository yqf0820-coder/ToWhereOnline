# 一路向哪 — 产品需求文档 (PRD)

## 产品概述

一路向哪（ToWhereOnline）是一款旅行日记 Web 应用，用户可以为去过的城市创建记录、上传照片、浏览时间轴、查看 AI 生成的回忆。

当前版本：MVP 迭代中 | 平台：Web (桌面端为主，移动端适配中)

## 四、技术债务

- Cesium 构建体积过大（~6MB），考虑 CDN 外链或动态 import
- 无测试覆盖
- 无 CI/CD
- 无错误监控（Sentry 等）
- 移动端尚未完整适配
